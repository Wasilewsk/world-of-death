# World of Death

A tactical strategy and survival game built with the Nonvisual Gaming Toolkit (NVGT). Lead your nation through global conflict, balance diplomacy and warfare, and build a robotic empire.

## How to Run from Source

To run **World of Death** directly from the source code, follow these steps:

1. **Install NVGT:** Ensure you have the [NVGT Engine](https://nvgt.dev/) installed on your system.
2. **Clone the Repository:** 
   ```bash
   git clone https://github.com/wasilewsk/world-of-wore.git
   ```
3. **Open Terminal:** Navigate to the project directory in your terminal or command prompt.
4. **Run the Game:**
   ```bash
   nvgt main.nvgt
   ```

## Key Features

*   **Tactical Combat:** Multi-choice battle system with power and resource management.
*   **Espionage & Hacking:** Siphon resources and steal blueprints via the Cyber Warfare console.
*   **3D Audio Gathering:** An interactive forest minigame with stereo panning and radar guidance to find wood and metal.
*   **Automation:** Build Gathering, Stealing, and War robots to manage your nation while you focus on high-level strategy.
*   **Dynamic World:** Global tension rises over time, triggering pandemics, economic booms, and the emergence of new futuristic nations.
*   **Live Updates:** Compiled versions include a built-in checker to download the latest releases from GitHub.

## Controls

*   **World Map:** Left/Right to travel, C for Combat, D for Diplomacy, S for Steal, H for Hack.
*   **Gathering:** WASD to move, Space for coordinates/direction.
*   **Menus:** Up/Down to navigate, Enter to select, Escape to Pause/Save.
